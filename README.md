# Boutique Project
A full-stack web application designed to manage boutique orders, expenses,and business records efficiently. Built to solve real-world billing and data management needs for small businesses.
##  What This Project Does

This is a business management system that helps boutique owners:
- Manage customer orders
- Track expenses
- Maintain structured records in a database
- View data in clean tabular format

This project demonstrates real-world full-stack development,
from UI to backend logic and database integration.
